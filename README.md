# bcgamejam2019
BC Game Jam

## Requirements
NodeJS 10: https://nodejs.org/en/download/ 

## Setup
Clone the project:

```bash
git clone https://github.com/ivanincanada/bcgamejam2019
```

Enter the new directory:

```bash
cd bcgamejam2019
```

Update the dependencies:
```bash
npm install
```

Start the server:
```bash
npm start
```
Check your game! http://localhost:3000
